# MarkFlow

AI Context Compression Engine

MarkFlow transforms noisy PDF documents into structured, AI-friendly Markdown while measuring compression efficiency, token reduction, and context quality.

## Overview

Large Language Models perform best when provided with clean, structured information. Raw documents often contain formatting issues, unnecessary content, and token inefficiencies.

MarkFlow processes uploaded documents through a structured pipeline:

PDF → Extract → Clean → Structure → Markdown → Analytics

## Features

* PDF document processing
* Markdown conversion
* Context compression analysis
* Token reduction metrics
* Character reduction metrics
* Download generated Markdown
* Interactive analytics dashboard

## Tech Stack

### Backend

* Python
* FastAPI

### Frontend

* React
* JavaScript

### AI Infrastructure

* Model Context Protocol (MCP)
* Document Processing Pipeline

## How It Works

1. Upload a PDF document
2. Extract document content
3. Clean noisy formatting
4. Structure information
5. Generate Markdown output
6. Analyze compression efficiency
